# NBA Dashboard Guide
I'm creating this dashboard to have an interactive tool, where I can enter data to generate information that will allow me to make more accurate predictions. This will also elimiate the need for me to collect data in a spread sheet on a day-to-day basis, then run code to generate predictive values.

## Things for Dashboard to Do
1. Display a dataset given to it and filter when values are specified.
2. Generate various types of plots to compare teams.
   1. **Compare teams on a polar graph**.
   2. Use bar graph to compare scoring distribution (don't have bins dodge each other).
      1. Example: https://gallery.shinyapps.io/template-nba-dashboard/
3. Simulate Basic Statistical Test
   1. T-Test
   2. Chi-Squared
4. Predict the outcome of specific events.
   1. Game Outcome
   2. Player Statistics
   3. Game Over/Under
   4. If Teams Cover the Spread
